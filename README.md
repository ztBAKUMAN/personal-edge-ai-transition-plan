# personal-edge-ai-transition-plan

> 从现代 C++ 到边缘 AI：个人工程能力转型实验室

面向已有 C++ / Linux 基础的工程师，从"知识点 Demo"升级为覆盖原生 C++ 工程底座、Linux 系统工程、性能调优、AI 推理部署主线、工业集成项目的长期学习工程。

## 五层能力架构

```
Phase 01: 原生 C++ 工程底座
Phase 02: Linux 系统工程与通信
Phase 03: 性能基础与数据通路
Phase 04: 通用 AI 推理部署主线
Phase 05: 数据库与中间件
Phase 06: 进阶主题
edge_core: 可复用组件库（持续沉淀）
```
